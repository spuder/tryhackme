# Updog

https://github.com/sc0tfree/updog


updog is a replacement for python http server. 

Instead of `python3 -m http.server` just run updog. 

It supports indexing so you don't need to know the filename, you can brows to it. 

## Install

pip install updog

or

pipx install updog

## Usage


```bash
updog
```

or

```bash
updog -b 192.168.1.10 -p 8080
```

```bash
updog --hide-base-path
```
