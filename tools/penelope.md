# penelope 

penelope is a reverse shell

https://github.com/brightio/penelope

Like netcat but more modern. 


## Usage
```
wget https://raw.githubusercontent.com/brightio/penelope/refs/heads/main/penelope.py && python3 penelope.py
```


Then on the target use the reverse shell generator

https://www.revshells.com/
